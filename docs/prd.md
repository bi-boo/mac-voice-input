# mac 语音输入法 - 产品需求文档

## 目标与背景
本项目旨在为 macOS 用户提供一个轻量、高效、美观的原生语音输入辅助工具。通过集成火山引擎 ASR（大模型异步识别），解决系统自带语音输入识别率低、灵敏度不足或特定场景下的兼容性问题。

## 核心功能
*   **全局录音识别**：通过快捷键触发，实时采集麦克风音频并显示流式识别结果。
*   **智能文本插入**：识别完成后，将结果自动粘贴至当前活跃的输入框。
*   **原生 HUD UI**：符合 macOS 设计规范的悬浮窗口，支持毛玻璃效果和 SF Symbols。
*   **自定义配置**：支持用户可视化配置 API 密钥和全局快捷键。

## 用户流程
### H1: 语音录入
#### H2: 启动录音
*   用户按下全局快捷键（默认 `⌘+Shift+V`）。
*   应用弹出 HUD 窗口，显示“正在聆听...”。
#### H2: 识别展示
*   流式显示识别中的文字。
*   提供“暂停/恢复”和“取消”操作。
#### H2: 完成插入
*   用户点击“完成并插入”或按下回车键。
*   应用隐藏窗口，将文本写入剪贴板并模拟 `⌘+V` 粘贴至原应用。

### H1: 设置与管理
#### H2: 偏好设置窗口
*   用户通过菜单栏点击“设置”进入。
*   窗口采用原生 macOS Tab 风格，包含“通用”和“快捷键”两个标签页。
#### H3: 通用设置
*   配置火山引擎 API 密钥（AppID 和 Access Token）。
*   查看并刷新麦克风和辅助功能权限状态。
#### H3: 快捷键设置
*   自定义“按住说话”和“按下说话”的全局快捷键。
*   界面符合 macOS 原生对齐规范：标签居左，控件居右。
