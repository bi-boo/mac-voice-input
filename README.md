# 🎙️ Mac 语音输入法

> 一款轻量级 macOS 原生语音输入工具，让你用说的代替打字。

基于火山引擎 ASR（豆包大模型）实现实时语音转文字，支持自定义快捷键、断句灵敏度调节等功能。

## ✨ 功能特性

- **实时转写**：边说边显示识别结果，无需等待
- **高精度识别**：支持豆包 2.0 大模型，识别准确率高
- **智能断句**：自动添加标点符号，支持语义顺滑（去除口语废话）
- **灵活快捷键**：支持「按住说话」和「按下切换」两种模式
- **原生体验**：纯 Swift 开发，轻量无依赖，与 macOS 完美融合

## 🚀 快速开始

### 方式一：直接下载（推荐）

1. 前往 [Releases](../../releases) 页面，下载最新的 `语音输入法.dmg`
2. 打开 DMG，将应用拖入 Applications 文件夹
3. 首次打开应用，在菜单栏点击图标 → **设置**
4. 填入你的 **App ID** 和 **Access Token**
5. 授予麦克风和辅助功能权限
6. 开始语音输入！

### 方式二：自行编译

```bash
# 克隆仓库
git clone https://github.com/bi-boo/mac-voice-input.git
cd mac-voice-input

# 编译运行
./build-native.sh
open build/语音输入法.app
```

### 获取 API 密钥

本应用使用 **火山引擎语音识别服务**（豆包大模型），需要您自行申请：

1. 注册 [火山引擎](https://www.volcengine.com/) 账号
2. 进入 [语音技术控制台](https://console.volcengine.com/speech/app)，创建应用
3. 获取 **App ID** 和 **Access Token**，填入应用设置页面

> 💡 火山引擎提供免费额度，个人日常使用完全够用

## ⌨️ 使用方法

两种模式**同时可用**，根据场景选择：

| 模式         | 快捷键（默认）            | 操作说明                   |
| ------------ | ------------------------- | -------------------------- |
| **按住说话** | `fn + Control`            | 按住说话，松开自动粘贴     |
| **按下切换** | `Ctrl + Option + Cmd + 6` | 按下开始录音，回车确认输出 |
| **取消录音** | `ESC`                     | 取消当前录音               |

快捷键可在设置页面自定义。

## ⚙️ 高级配置

编辑 `asr_settings.json` 可调整识别参数：

| 参数                 | 说明                                   | 推荐值 |
| -------------------- | -------------------------------------- | ------ |
| `enable_nonstream`   | 二遍识别，显著提升准确率               | `true` |
| `enable_ddc`         | 语义顺滑，去除"呃、啊"等语气词         | `true` |
| `enable_punctuation` | 自动添加标点                           | `true` |
| `end_window_size`    | 断句灵敏度（毫秒），值越大越不容易断句 | `800`  |

## 📋 系统要求

- macOS 12.0 (Monterey) 或更高版本
- Apple Silicon (M1/M2/M3) 或 Intel Mac
- 需授予麦克风和辅助功能权限
